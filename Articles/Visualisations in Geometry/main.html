Visualisations in Geometry

Otis Laundon
(March 16, 2026)
Abstract

In this project we explore various concepts in the area of differential geometry, topology, and Lie groups through interactive visualisations. Generalising the visualisation technique of domain colouring (typically seen in complex analysis), we use unordered pairs of colours, visualised through checkerboard patterns, which allows for a wider class of continuous functions to be visualised than by standard domain colouring. This technique, combined with standard computer graphics techniques, is then used to create interactive visualisations of the Lie groups S⁢O⁢(3) and S⁢U⁢(2), their left, right, and conjugation self actions, and the double covering of S⁢O⁢(3) by S⁢U⁢(2). We also show visually that the fundamental group of S⁢O⁢(3) is ℤ/2⁢ℤ. Both a PDF and an interactive online version of this document are available. Each version contains the same content, however the online version allows for real-time interaction with the visualisations presented.
