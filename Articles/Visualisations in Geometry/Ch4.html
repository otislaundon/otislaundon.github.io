Chapter 4 Visualising SU⁢(2)

Recall that the special unitary group SU⁢(2) is

SU⁢(2)={A∈GL⁢(2,ℂ)|A⁢A†=𝕀,det(A)=1}

where A† is the conjugate transpose of A. More concretely, it is the set of 2×2 complex matrices

(αβ−β¯α¯)

with α,β∈ℂ such that |α|2+|β|2=1 [HAL15], thus making SU⁢(2) homeomorphic to S3. This should be familiar from our earlier discussion of Möbius transformations, where the group Rot⁡(Σ) was shown to be a quotient of this group of matrices. As seen in 6.6, SU⁢(2) is isomorphic to the set of unit quaternions ℍ1.

The Lie algebra 𝔰⁢𝔲⁢(2) is the set of 2×2 skew-Hermitian traceless matrices

𝔰⁢𝔲⁢(2)={A∈GL⁢(2,ℂ)|A†=−A,tr⁢(A)=0},

which is spanned by

u1=(i00−i),u2=(01−10),u3=(0ii0). (4.1)

Topologically, S3 can be obtained by the set {(u,v)∈ℂ2||u|2+|v|2=1}, by the identification of the boundary of a closed 3-ball, or by the one-point compactification of ℝ3. The way which we choose here to visualise it is by a 3-ball of radius π, much like we did for SO⁢(3), since a subset of ℝ3 is easier to conceptualise and visualise than a subset of ℂ2≅ℝ4. We use the exponential map exp:𝔰⁢𝔲⁢(2)→SU⁢(2) to identify the points in B0⁢(π) with elements of SU⁢(2). Note the difference with the ball of radius π being used to visualise SO⁢(3), where we had to identify antipodal points on the boundary, whereas when visualising the 3-sphere in this way we instead need to identify the whole boundary of the 3-ball, collapsing the boundary down to a single point.

When using the basis (4.1) for 𝔰⁢𝔲⁢(2), the exponential map (with image written as a unit quaternion as in 6.7) is

exp⁡(𝐱)=cos⁡(‖𝐱‖)+sin⁡(‖𝐱‖)⁢x⁢i+y⁢j+z⁢k‖𝐱‖

for 𝐱=x⁢u1+y⁢u2+z⁢u3∈𝔰⁢𝔲⁢(2). This function is injective for all 0≤‖𝐱‖<π, and is equal to (−1000) when ‖𝐱‖=π. The exponential map has an inverse (the logarithm) on exp⁡(B0⁢(π)) which gives us a maximal dense chart of SU⁢(2), which is defined on SU⁢(2)\(−1000).

This is analogous to the 2-dimensional case, where we can parameterise the 2-sphere (minus one point) by “wrapping a disc around it”. Figure 4.1 shows how a disk can be wrapped around a sphere, using a domain colouring to show the mapping. The sphere has a faithful colour scheme applied to it, and the disk is mapped onto the sphere such that points with the same colour match up.

In figure 4.1, a continuous faithful colour scheme was put on the 2-sphere. However, as discussed in section 2.3, it is not possible to put a continuous faithful colour scheme on the 3-sphere, since it cannot be embedded into ℝ3. With this in mind, we construct a faithful double colour scheme for S3, so that we can use it to visualise maps into S3.

Our double colour scheme for S3 will have the property that one of the poles (−1000) is just the pair [white,white], and the opposite pole (1000) is the pair [black,black]. This makes the double colour scheme we construct for S3 analogous to the colour scheme for S2 shown in 4.1, and also similar to 3.5, the points closest to the identity in SU⁢(2) will have darkest colours, with the identity itself being black.

Definition 4.0.1.

We define the double colour scheme τS3:S3→Colu2 by its HSV components, given by

τS3⁢(w,x,y,z)=[HSV⁡(h−d⁢h,s,v),HSV⁡(h+d⁢h,s,v)]

where

h =x+14
s =y+12⁢(1−(1−w2)4)
v =1−w2
d⁢h =z+19

and x,y,z,w∈ℝ such that w2+x2+y2+z2=1.

Note that we do indeed have that (−1000) has 0 saturation and 1 value (making it white), and both colours are equal since d⁢h=0, and also that (1000) has 0 value and both colours are equal. Also, each component h,s,v is in the range [0,1]⊂ℝ since w,x,y,z all range from −1 to 1, so they are valid HSV components. Also 0≤d⁢h≤2/9<1/4, meaning that we will not have the issue of distinct values of h giving rise to a pair of colours with hues h−d⁢h and h+d⁢h which are the same.

Proposition 4.0.1.

The double colour scheme τS3 is continuous and faithful.

Proof.

Continuity is clear from construction. Let 𝐚1=(w1,x1,y1,z1) and 𝐚2=(w2,x2,y2,z2). Denote

τS3⁢(𝐚1)=[HSV⁡(h1−d⁢h1,s1,v1),HSV⁡(h1+d⁢h1,s1,v1)]
τS3⁢(𝐚2)=[HSV⁡(h2−d⁢h2,s2,v2),HSV⁡(h2+d⁢h2,s2,v2)].

We can assume d⁢h1,d⁢h2≥0 since our pairs of colours are unordered. Suppose τS3⁢(𝐚1)=τS3⁢(𝐚2). Then v1=v2⟹w1=w2. Since v1=v2=0 if and only if w1=w2=1, and there is only a single point in S3 with w=1, we can assume v1=v2≠0. Then, since v≠0, saturation component is well defined (recall Remark 2.1.1). Therefore s1=s2 which implies y1=y2. Also, the saturation is only 0 when y=−1 (which only happens at a single point) or w=−1 (which also only happens at a single point). Therefore, we can assume that the saturation is non-zero, and so the hue component is well-defined. We then have one of 2 cases:

Case 1: h1−d⁢h1=h2−d⁢h2 and h1+d⁢h1=h2+d⁢h2 giving that

h1=h2+(d⁢h2−d⁢h1)=h2−(d⁢h2−d⁢h1)

which implies that d⁢h2−d⁢h1=0 and so d⁢h2=d⁢h1 and h2=h1.

Case 2: h1−d⁢h1=h2+d⁢h2 and h1+d⁢h1=h2−d⁢h2 giving that

h1=h2−d⁢h2−d⁢h1=h2+d⁢h2+d⁢h1

and so (d⁢h2+d⁢h1)=−(d⁢h2+d⁢h1) giving that d⁢h2=−d⁢h1, but since both d⁢h2,d⁢h1 are non-negative, they must both be 0. Hence d⁢h2=d⁢h1 and h2=h1. In either case we have that h1=h2. The hue component is chosen to only take values in [0,1/2] since the hues 0 and 1 are the same, so we only use half of the range of hues to ensure that distinct x values give rise to distinct hues, and so x1=x2.

Since d⁢h1=d⁢h2, and d⁢h values in the range [0,2/9] give distinct colour pairs (when the value and saturation are nonzero), we get that z1=z2. Therefore, 𝐚1=𝐚2 and so τS3 is injective. ∎

In figure 4.2 we show the double colour scheme τS3 applied to a subset of B0⁢(1)¯ consisting of 3 axis-aligned disks. It is visually apparent that this double colour scheme is continuous, constant on the boundary of B0⁢(1)¯, and otherwise faithful. Therefore, this double colour scheme visually describes the identification of B0⁢(1)¯ which gives rise to the topology of S3 by identifying points with the same unordered pair of colours assigned to them.

The disks seen in figure 4.2 are chosen as a subset of B0⁢(π)¯ on which to visualise the colour scheme τS3, because showing the surface of the ball would be uninteresting as the double colour scheme is constant white over the surface (because exp:𝔰⁢𝔲⁢(2)→SU⁢(2) is constant for vectors of length π).

We remark that the 3-sphere is often visualised through stereographic projection onto ℝ3, which maps the set S3\{(−1000)} bijectively to ℝ3. One benefit of visualising it in this way is that geodesics in the 3-sphere map to circles in ℝ3 under stereographic projection, whereas when using the ball with boundary identified, these circles become distorted. However, one benefit of using the closed ball rather than all of ℝ3 is that both the closed ball and S3 are compact but ℝ3 is not, and so the visualisation using the closed ball conveys this topological property better than if we use stereographic projection. A ball can also be drawn in such a way that every point is in view at once, whereas this is not the case when rendering ℝ3 with 3D perspective, as there will always be points “behind” the viewer.

4.1 Covering map SU⁢(2)→SO⁢(3)

In this section, we provide the definition, and a visualisation the two-fold covering map Spin:SU⁢(2)→SO⁢(3). The material in this section closely follows Appendix A from [NAB10].

If we think only topologically first, we know that SU⁢(2) is homeomorphic to S3, and SO⁢(3) is homeomorphic to ℝ⁢P3. If we take the quotient map Q:S3→ℝ⁢P3 for the equivalence relation that identifies antipodal points in S3, then this is a two-to-one covering map of ℝ⁢P3 by S3. We are interested in a map that is more than just a two-fold covering map though, since SU⁢(2) and SO⁢(3) are Lie groups, we really want our covering map to be a group homomorphism too. This condition is fulfilled by the spinor map.

Definition 4.1.1.

Let U=(w+i⁢xy+i⁢z−y+i⁢zw−i⁢x)∈SU⁢(2). The spinor map Spin:SU⁢(2)→SO⁢(3) is defined by

Spin⁢(U)=(1−2⁢y2−2⁢z22⁢x⁢y+2⁢z⁢w2⁢x⁢z−2⁢y⁢w−2⁢x⁢y+2⁢z⁢w1−2⁢x2−2⁢z22⁢y⁢z+2⁢x⁢w2⁢x⁢z+2⁢y⁢w2⁢y⁢z−2⁢x⁢w1−2⁢x2−2⁢y2)

Definition 4.1.1 is the same as that given in [NAB10], only rewriting diagonal entries using the fact that w2+x2+y2+z2=1.

Theorem 4.1.1.

[NAB10] The spinor map Spin:SU⁢(2)→SO⁢(3) is a smooth, surjective group homomorphism with kernel ±(1001).

With our identification 6.6 of elements of SU⁢(2) and unit-quaternions (wxyz), we see that the Spin map has kernel ±(1000). This means that both the centre of the 3-ball description of the 3-sphere, and it’s entire boundary together form the kernel of Spin.

Putting double domain colouring to further use, let’s see what a double domain colouring for Spin:SU⁢(2)→SO⁢(3) looks like using the double colour scheme τ0 3.4 for SO⁢(3). Figure 4.3 shows the double domain colouring

B0⁢(π)¯→expSU⁢(2)→SpinSO⁢(3)→τ0Colu2

for B0⁢(π)¯.

In Figure 4.3 we see the kernel of Spin as the points which are coloured black, as under the double colour scheme τ0, the identity is the only point coloured black. We therefore see that both the point (1000), which is the point at the centre of our ball, and the point (−1000), which is the entire boundary of the ball, form the kernel, since these are the parts that are coloured black.

The Lie algebras 𝔰⁢𝔬⁢(3) and 𝔰⁢𝔲⁢(2) are isomorphic [HAL15]. This is because our basis 4.1 for 𝔰⁢𝔲⁢(2), when scaled down by a half to give 12⁢u1,12⁢u2,12⁢u3 satisfy the relations

[12⁢u1,12⁢u2]=12⁢u3,[12⁢u2,12⁢u3]=12⁢u1,[12⁢u3,12⁢u1]=12⁢u2,

and also, our basis 3.3 Ux,Uy,Uz satisfy the relations

[Ux,Uy]=Uz,[Uy,Uz]=Ux,[Uz,Ux]=Uy.

By skew-symmetry of the Lie bracket, these completely determine all of the relations satisfied [HAL15], and so the map

Ux↦12⁢u1,Uy↦12⁢u2,Uz↦12⁢u3

is a Lie algebra isomorphism. We in fact have that

Spin⁢(exp⁡(12⁢(α1⁢u1+α2⁢u2+α3⁢u3)))=exp⁡(α1⁢Ux+α2⁢Uy+α3⁢Uz)

where the exponential on the left hand side is 𝔰⁢𝔲⁢(2)→SU⁢(2) and the exponential on the right hand side is 𝔰⁢𝔬⁢(3)→SO⁢(3). Then, using this isomorphism of Lie algebras, which is compatible with the Spin map, we combine our visualisation for SO⁢(3) which uses the ball of radius π and our similar visualisation for SU⁢(2) which also uses a ball or radius π. Take the ball in 𝔰⁢𝔬⁢(3) and shrink it by a factor of 12 and then embed it into the ball of radius π in 𝔰⁢𝔲⁢(2). The result is shown in Figure 4.4.

In order to make Figure 4.4 interactive, we use the fact that SO⁢(3) is locally isomorphic to SU⁢(2). One way to see this is that SU⁢(2) is isomorphic to Spin⁢(3), the universal covering group of SO⁢(3) [NAK03, TAY54]. Through this local group isomorphism, we can treat sufficiently small elements of SO⁢(3) as elements of SU⁢(2). Whilst interacting with Figures 3.4, 3.7, 3.8, by rotating the cube on the right hand side in order to change θ⁢𝐮, from one frame to the next a rotation R by a small angle was being composed with the existing rotation b=θ⁢𝐮. We extend this procedure to interactively move around an element b~∈SU⁢(2) as follows:

Start with b=e∈SO⁢(3) and b~=e∈SU⁢(2) so that Spin⁢(b~)=b. Then on each time-step of the interactive visualisation compose b with the rotation R defined by the mouse movement of the user (which will generally be small), and also compose b~ with R~, where R~ is the element of SU⁢(2) corresponding to R under our local isomorphism of SO⁢(3) and SU⁢(2) near the identity. Considering the path in SO⁢(3), with base point at the identity, that b takes as we interact with it, the point b~ will always be the corresponding point on the lifted path in SU⁢(2) which has base point at the identity of SU⁢(2).

Another way of looking at this is that as b moves along the path with velocity controlled by the user’s mouse input, it will always have some velocity in the tangent space to b. We can identify the tangent space to b in SO⁢(3) with the tangent space to b~ through the differential of the Spin map. Then the velocity of b determines the velocity of b~, which accordingly moves along the lifted path.

In this way, we can interact with SU⁢(2) in a very natural way. We can reach any element of SU⁢(2) by a series of compositions of small rotations, each lifted to SU⁢(2). When rotating the cube on the right hand side of 4.4, we modify b and b~ according to the process outlined above.

4.2 Left, Right, and Conjugation Actions on SU⁢(2)

Just as we did for SO⁢(3) in section 3.2, we can demonstrate the group structure of SU⁢(2) by showing the left, right, and conjugation actions of SU⁢(2) on itself, each visualised through it’s effect on a collection of points {ai}1≤i≤k in SU⁢(2).

Figures 4.5, 4.6, 4.7, show the left, right, and conjugation actions of SU⁢(2) on itself by the element b~, which is the lift of the point b as determined by the procedure described in the previous section. The points b and b~ are denoted by blue and red dots respectively. Each of these visualisations has points {ai}1≤i≤k initially randomly selected on spheres of increasing radius. The points {ai} are drawn with a checker pattern which has colours assigned using the double colour scheme τS3.

In the interactive versions of Figure 4.5, the visualisation of Lb~, as we change b~ there is a translational motion of the points in SU⁢(2) in direction 𝐮, and twisting motion anticlockwise around 𝐮. This is very similar to what was observed in 3.4. As we change b=θ⁢𝐮, the twisting motion that takes place in SU⁢(2) in figures 4.5 is only by half of the angle that we change b by, and so the cube on the right hand side must make 2 full rotations before the points on the left hand side return to their original positions. We have at the same time that as b makes one full rotation, the point b~ moves to the outer part of the ball, which corresponds to the hemisphere {(w,x,y,z)∈S3|x<0}. Then after another full rotation of b, b~ and b are together at the identity again.

All of these properties apply to the visualisation in Figure 4.6 of the right action by b~ too, only the twisting action happens in the opposite (clockwise) direction around 𝐮. The translational motion is the same for both the left and right actions because composition of rotations around the same axis is commutative since the set of rotations around a fixed axis form a one-parameter family which is a subgroup of SU⁢(2) isomorphic to the circle group S1, which is commutative.

In the case of the conjugation action in 4.7, we have that Cb~=Lb~∘Rb~−1. This causes the translational motions of Lb~ and Rb~ to cancel out, and the twisting motions which are rotations by half angles in opposite directions to sum to be a rotation by full angle of the change in rotation being applied on the right hand side. The result is that the transformation applied to the points on the left hand side is a rotation, and moreover, it is the same as the rotation of the cube on the right hand side, around axis 𝐮 by angle θ. This shows that the adjoint representation of SU⁢(2) factors through SO⁢(3).

4.3 Fundamental Group of SO⁢(3)

In this section we will visualise the fact that the fundamental group of SO⁢(3) is ℤ/2⁢ℤ.

We know from Section 4.1 that SU⁢(2) is the 2-fold cover of SO⁢(3), and gave a concrete description of the covering map Spin:SU⁢(2)→SO⁢(3) as well as visualised the covering map.

We will be using the fact that paths in SO⁢(3) can be lifted to paths in SU⁢(2). This lifting of paths in SO⁢(3) to paths in SU⁢(2) was the principle used for interacting with SU⁢(2) introduced for the visualisation in Figure 4.4.

A good way at getting at the fundamental group of a topological space is to look at the group of deck transformations of it’s universal covering space.

Definition 4.3.1.

[HAT02] For a covering space π:X~→X, the isomorphisms f:X~→X~ (such that π∘f=π) are called deck transformations.

The group of deck transformations of a simply-connected locally path-connected covering space X~ is isomorphic to the fundamental group π1⁢(X) (the base point is unimportant due to connectedness of X) [HAT02]. In our case, SU⁢(2) is the simply connected, locally path connected covering space of SO⁢(3), and so to find the fundamental group of SO⁢(3) it suffices to find the group of deck transformations.

The Spin map, which is the covering map SU⁢(2)→SO⁢(3) has kernel

±𝕀⊂SU⁢(2)
= ±(1000)⊂ℍ1,

which means that when we lift a closed path α:I→SO⁢(3) with base point x0 to a path α~:I→SU⁢(2) with base point x0~ with π⁢(x0~)=x0, we get that α~⁢(1) must be ±x0. As there are only 2 possible values for α~⁢(1) here, the group of deck transformations can have at most 2 elements. To see that it has at least 2 elements we only need to show that there exists a loop α:I→SO⁢(3) that lifts to a non-closed path in SU⁢(2), giving a non-trivial element of the group of deck transformations, and therefore a non-trivial element of π1⁢(SO⁢(3)). In figure 4.8 we see an example of a path α which is closed in SO⁢(3) which lifts to a non-closed path in SU⁢(2). This shows visually the existence of a non-trivial element of the group of deck transformations. Any group with 2 elements is isomorphic to ℤ/2⁢ℤ, and so π1⁢(SO⁢(3))≅ℤ/2⁢ℤ.

If we concatenate the path α with itself, to get α∗α=α2, then we will have that α2~ is a closed loop. This is because for a nontrivial paths α:I→SO⁢(3) with base point p, the lifted path with base point q∈SO⁢(2) has α~⁢(0)=q and α~⁢(1)=−q. So running through any path twice causes our lifted path to run from q to −q and then back to q, making it guaranteed to be a closed loop. Since SU⁢(2) is simply connected, every closed loop is contractible. In figure 4.9 we see this in practice for the path α from 4.8. As t ranges from 0 to 12, the point α⁢(t) returns to it’s starting point but the point α~⁢(t) now lies in the outer part of the ball (i.e. the hemisphere of S3 with x<0). Then as we make a second pass along α, with t going from 12 to 1, we see that α~⁢(t) returns to the starting point along with α.

This fact can also be visually demonstrated through the Dirac belt trick. To perform this demonstration, take a belt (the kind used to hold up trousers) or some other long strip of material. The belt represents a path in SO⁢(3) where each t∈[0,1], corresponding to a point along the length of the belt, maps to the orientation in space of the surface of the belt near point t. One can introduce a twist of 720 degrees, and then, keeping the orientation of the endpoints of the belt fixed, undo this twist and straighten out the belt, thus showing that a path in SO⁢(3) which makes 2 full rotations is homotopic to the constant path at the identity. This is described and illustrated in [STA10].
